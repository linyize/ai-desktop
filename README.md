# ai-desktop
Lightweight AI-native desktop shell for Linux — chat-first interaction, local-first AI, no clutter.

🪟 AI sidebar dock for Linux desktops (GNOME/KDE/XFCE)


💬 Natural language commands: "open browser", "find photos", "backup files"


🤖 Local-first: runs Qwen/LLama locally via llama.cpp, no cloud needed


⚡ Global hotkey trigger (Super+Space) — instant AI access


🔒 Sandbox execution for safety
