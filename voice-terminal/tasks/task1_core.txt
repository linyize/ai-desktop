### 2. audio_utils.py
Utility functions for audio processing:
- pcm16_to_wav(pcm_bytes: bytes, sample_rate: int = 16000) -> bytes: Wrap raw PCM16 mono bytes in a WAV header, return complete WAV bytes.
- wav_to_pcm16(wav_bytes: bytes) -> bytes: Strip WAV header, return raw PCM16.
- normalize_volume(pcm_bytes: bytes, target_db: float = -3.0) -> bytes: Simple peak normalization.
- save_audio(pcm_bytes: bytes, path: str, sample_rate: int = 16000): Save PCM16 bytes as WAV file.
- read_audio(path: str) -> tuple[bytes, int]: Read WAV file, return (pcm16_bytes, sample_rate).
